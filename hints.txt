To locally run a the docker image and open a bash sell
change DOCKERFILE so there is no entrypoint and the CMD is: CMD [ "/bin/bash" ]
build the image with name "tax-form-ocr-docker"
then run: docker run -it tax-form-ocr-docker


run bash in container from command line
docker exec -t -i mycontainer /bin/bash
docker run -it --entrypoint /bin/bash a2330cc62cfe

To build the docker image and deploy it:
docker build -t tax-form-ocr-docker .
aws ecr get-login-password --region us-east-1 | docker login --username AWS --password-stdin 206046575545.dkr.ecr.us-east-1.amazonaws.com/tax-form-ocr
docker tag  tax-form-ocr-docker:latest 206046575545.dkr.ecr.us-east-1.amazonaws.com/tax-form-ocr:latest
docker push 206046575545.dkr.ecr.us-east-1.amazonaws.com/tax-form-ocr:latest
